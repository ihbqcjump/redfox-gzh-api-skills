# 根据作品地址获取公众号作品（优质库）

## 1. 简介

通过 RedFox API 根据公众号文章链接获取文章的完整详情，包括正文内容、标题、作者、发布时间、阅读/点赞/在看等互动数据、原创标识、封面图等。适用于已知文章 URL 需要提取全文内容或详细数据的场景。

## 2. 功能特性

- 根据文章 URL 获取完整详情
- 返回正文内容（全文）
- 返回阅读/在看/点赞/评论/收藏/分享/打赏等完整互动数据
- 标识原创、显示原创账号和原文链接
- 返回封面图、发文位置、账号分类等元信息
- 支持 JSON 原始输出和格式化输出

## 3. 环境变量

| 环境变量 | 说明 | 是否必填 |
| -------- | ---- | -------- |
| `REDFOX_API_KEY` | 红狐数据 API Key | 是 |

## 4. 使用指南

```bash
python scripts/query_article_detail.py --url <文章链接>
python scripts/query_article_detail.py --url <文章链接> --json
```

### 参数说明

- `--url`（必填）：公众号文章链接，如 `https://mp.weixin.qq.com/s/example`
- `--json`（可选）：输出原始 JSON 数据

```

作者: 科研云
分类: 时事新闻
发布时间: 2026-01-15 10:00:00

阅读: 5.0w | 在看: 1200 | 点赞: 3500
评论: 280 | 收藏: 500 | 分享: 150 | 打赏: 30
链接: https://mp.weixin.qq.com/s/example
文章位置: 第1条（0=头条）

摘要: 这是文章摘要内容

正文（前500字）: 这是文章正文内容...
```
