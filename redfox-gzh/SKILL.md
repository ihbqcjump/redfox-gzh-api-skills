# RedFox 公众号全自动分析

一个关键词 → 自动跑完 6 个 API → 输出完整分析报告。无需手动分步操作。

## 环境变量

| 环境变量 | 说明 | 是否必填 |
| -------- | ---- | -------- |
| `REDFOX_API_KEY` | 红狐数据 API Key | 是 |

## 用法

```bash
python scripts/redfox.py "关键词"
python scripts/redfox.py "关键词" --sort hottest    # 按热度排序
python scripts/redfox.py "关键词" --sort latest     # 按最新排序
python scripts/redfox.py "关键词" --top 5           # 每类显示 Top 5
python scripts/redfox.py "关键词" --json            # 输出结构化 JSON
```

只需一个关键词，其余全自动。

## 自动化流程（6 个 API 串联）

脚本接收关键词后，自动按顺序调用 6 个 API：

```
输入: "人工智能"
  │
  ├─① searchUser ──────→ 搜索相关公众号账号（共 N 个）
  ├─② searchArticle ────→ 搜索相关文章（共 N 篇）
  ├─③ queryUser ────────→ 获取 Top1 账号完整画像（认证/标签/简介/地区）
  ├─④ queryWork ────────→ 获取 Top1 文章正文 + 内容关键词云
  ├─⑤ queryArticleDetail→ 获取 Top1 文章 URL 详情（补充互动数据）
  └─⑥ queryAiMsgs ──────→ 搜索 AI 创作的相关内容
  │
输出: 完整分析报告
  ├─ [1] 相关账号概览（Top N 列表）
  ├─ [2] 头部账号深度画像（认证/分类/标签/简介/最近发文）
  ├─ [3] 相关文章概览（Top N 列表 + 互动数据）
  ├─ [4] 头部文章深度解析（正文节选 + 关键词云 + 全部互动指标）
  ├─ [5] AI 创作内容（标题/作者/互动数据/话题）
  └─ 数据概览总结
```

## 报告结构

输出的报告包含 5 个板块：

1. **相关账号概览** — 搜索到的公众号列表，含红狐指数、分类、认证
2. **头部账号画像** — 排名第一的公众号完整信息（认证状态、标签、简介、地区、最近发文）
3. **相关文章概览** — 搜索到的文章列表，含阅读/点赞/在看/分享等互动数据
4. **头部文章解析** — 排名第一的文章完整分析：正文节选、内容关键词云、全部互动指标、原创标识
5. **AI 创作内容** — 该话题下 AI 生成的文章，含阅读/点赞/话题标签

## 排序选项

- `default`（默认）：按相关性排序
- `latest`：按最新发布时间排序
- `hottest`：按阅读量排序（最热）

## 典型场景

**分析某话题的公众号生态**
```bash
python scripts/redfox.py "人工智能"
```

**看某话题最热的内容**
```bash
python scripts/redfox.py "量子计算" --sort hottest
```

**获取结构化数据做进一步处理**
```bash
python scripts/redfox.py "新能源" --json
```
